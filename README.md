✍🏻 Blog Posts
You can find my blog [here](https://blog.mgattozzi.dev) but I have a few posts
that I think are worth your time:

- [Oxidizing the technical interview](https://blog.mgattozzi.dev/oxidizing-the-technical-interview/):
  by far my most popular article that made it to places I wish it hadn't like the front page of
  Hacker News! If you want a laugh and to curl up in terror about Rust it's worth a read!
- [Rust and it's Orphan Rules](https://blog.mgattozzi.dev/orphan-rules/): Rust
  has rules that unfortunately were not made to be broken. It includes some
  possible solutions if you run into issues
- [Package Managers for Programmers: Theory, Design, and
  Tradeoffs](https://blog.mgattozzi.dev/package-managers-for-programmers/):
  I think package managers are cool but I don't think everyone understands how
  their versions are *chosen* and this post dives into the theory in an
  accessible way because CS doesn't need to be hard.
